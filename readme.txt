Procedimento para conseguir rodar o teste de Inscrição de Cuso com sucesso.
1 - Pré-Requisitos
   - Node.js (versão Versão mais atualizada) - https://nodejs.org/en/download
   - NPM - É instalado junto com o Node
   - Cypress instalado globalmente ou no projeto
   - VSCODE como IDE - https://code.visualstudio.com/download

2 - Baixar o projeto para a máquina.
   - No terminal git bash digitar o comando git clone https://github.com/Kerbegneto/Desafio-Adapt.git.

3 - Abrir o projeto no VSCODE.
   -  No VSCODE ir em file + Open folder + e escolher o projeto que baixou na máquina.  

4 - Instalar o Cypress e rodar o teste.
   - No terminal do VSCODE digitar npm install Cypress.
   - Após instalar o Cypress digitar  npx Cypress open para abrir o Cypress.   
   - O Cypress abrira uma interface grafica onde você deverá escolher a opção E2E Testing.
   - Depois escolha o navegador que ira testar ele te da 3 opções CHrome, Edge e Electron e pressione o botão Start E2E Testing in e o nome do navegador que escolheu.
   - Depois em E2E specs irão aparecer todos os arquivos de testes que estão dentro do E2E do projeto, selecionar o arquivo inscricaocurso.cy.js e o teste irá iniciar.
